# 字符串
字符串相关方法

## camelize
横线转驼峰命名

名称|类型|必须|说明
---|---|---|---
str|String|true|待转换的字符串

```javascript
import tools from '@/tools'

tools.camelize(str)
```

## hyphenate

名称|类型|必须|说明

str|String|true|待转换的字符串

import tools from '@/tools'
