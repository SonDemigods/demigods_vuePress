## 开发环境
react的开发基于node运行，所以需要安装node环境。

### node
> [node下载地址](https://nodejs.org/en/)

选择适合当前系统的版本进行安装。
![class_001_01](./assets/class_001_01.png)

安装期间直接下一步就可以。
![class_001_02](./assets/class_001_02.png)

安装完成后验证node和npm安装情况。
```sh
node -v

npm -v
```

返回正确的版本号，说明安装成功。
