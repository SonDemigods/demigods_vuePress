## mobileLengthValidation

验证是否为正确手机号

| 名称  |  类型  | 默认值 | 必须 |     说明     |
| :---: | :----: | :----: | :--: | :----------: |
| value | String |  ‘ ’   | true | 被检测的数据 |

```js
import validations from '@/validations'

validations.mobileLengthValidation(value)
```
