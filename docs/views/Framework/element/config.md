# 配置说明

## 配置项

名称|类型|说明
---|---|---
cookieExpires|number|cookie存储时间，单位/天
baseUrl|object|ajax请求基础路径，dev：开发；pro：打包
homeName|string|首页路由的name
theme|string|主题名称
themeList|array|主题列表
sizeList|array|字号列表
topbar|boolean|顶部菜单开关
sidebar|boolean|侧边菜单开关

*注：当前主题不可动态切换
